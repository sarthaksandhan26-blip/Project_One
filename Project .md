## 📌 Project Summary

Developed an interactive Microsoft Excel dashboard to analyze bike-sharing demand using historical rental data from January 2011. This student project demonstrates hands-on experience in data cleaning, data transformation, pivot table analysis, and dashboard design.

## 📊 Dataset Overview

- **Dataset:** Bike Sharing Demand
- **Time Period:** January 2011
- **Total Records:** 610
- **Tool Used:** Microsoft Excel

## 🛠 Key Work Performed

- Cleaned and standardized raw data.
- Created temperature categories and time-based segments using Excel formulas.
- Built pivot tables and pivot charts to analyze rental patterns.
- Designed KPI cards to highlight important metrics.
- Developed a professional dashboard with visual summaries.

## 🔍 Key Insights

- Working Days represented 586 of 610 records.
- Clear weather was the most common condition (368 records).
- Afternoon and Evening showed higher bike rental activity.
- Cold temperatures (5–10°C) were the most frequent range.

## 💡 Skills Demonstrated

Microsoft Excel, Data Cleaning, IF/IFS Formulas, Pivot Tables, Pivot Charts, Conditional Formatting, KPI Dashboard Design, Data Analysis, Business Intelligence.
