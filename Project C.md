## 📌 Project Summary

Developed an advanced Microsoft Excel analytics dashboard to examine hourly bike-sharing demand and uncover patterns based on time, weather conditions, and user behavior.
This project demonstrates practical skills in data cleaning, formula creation, pivot table analysis, KPI development, and dashboard design. The dashboard highlights key performance indicators and visual insights that help explain how demand changes throughout the day and week.

## 📊 Dataset Overview

- **Dataset:** Bike Sharing Demand
- **Total Records:** 390 hourly observations
- **Tool Used:** Microsoft Excel
- **Analysis Type:** Hourly Demand Analytics Dashboard

### 📈 Key Metrics

- **Total Rentals:** 24,406
- **Average Rentals per Hour:** 62.6
- **Peak Hour Average:** 154.1 rentals
- **Best Day Average:** 76.4 rentals (Friday)
- **Registered User Share:** 91.1%
- **Casual User Share:** 8.9%
- **Lowest Weather Average:** 35.8 rentals (Light Rain/Snow)

## 🛠 Key Work Performed

- Cleaned and standardized raw hourly rental data.
- Created calculated fields for hour, weekday, and weather categories.
- Built pivot tables to summarize average rentals.
- Designed KPI cards to highlight performance metrics.
- Developed line charts, bar charts, and comparative visualizations.
- Applied conditional formatting and custom dashboard themes.

## 📈 Dashboard Sections

### ⏰ Average Hourly Demand
Analyzed rental trends across 24 hours to identify peak commuting periods.

### 📅 Weekend vs Weekday Average Demand
Compared usage patterns between weekdays and weekends.

### 👥 Casual vs Registered User Share
Visualized the contribution of each customer segment.

### 🌦 Average Rentals by Weather Condition
Measured the impact of weather on rental demand.

### 📆 Average Rentals by Day of Week
Compared average rentals across all seven days.

## 🔍 Key Insights

- Bike rentals show two strong peaks during morning and evening commute hours.
- The highest average hourly demand occurs at **5:00 PM**, reaching **154.1 rentals**.
- **Friday** records the highest average daily demand at **76.4 rentals**.
- **Registered users** contribute **91.1%** of total rentals.
- **Weekday demand** is higher than weekend demand.
- **Light Rain/Snow** results in the lowest average rentals, with **35.8 rentals**.
