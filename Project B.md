## 📌 Project Summary

Developed an advanced Microsoft Excel dashboard to analyze bike-sharing demand using historical rental data. This project demonstrates practical skills in data cleaning, data transformation, pivot table analysis, KPI creation, and professional dashboard design.
The dashboard focuses on rental behavior across temperature, humidity, user types, and demand levels, providing business insights into how environmental conditions affect bike-sharing usage.

## 📊 Dataset Overview

- **Dataset:** Bike Sharing Demand
- **Tool Used:** Microsoft Excel
- **Analysis Type:** Dashboard and KPI Reporting
- **Total Rentals:** 33,898
- **Registered Users:** 31,152
- **Casual Users:** 2,746
- **Average Rentals per Record:** 55.6
- **Peak Rentals in a Single Record:** 249
- **Registered User Share:** 91.9%
- **Standard Deviation:** 49.3

## 🛠 Key Work Performed

- Cleaned and standardized raw data.
- Created derived categories for temperature, humidity, and demand levels.
- Built pivot tables to summarize rental patterns.
- Designed KPI cards to highlight performance metrics.
- Developed pivot charts including bar charts and pie charts.
- Created a professional interactive dashboard layout.
- Applied conditional formatting and custom themes.

- Registered users account for 91.9% of total rentals.
- Casual users contribute only 8.1%.
- Cold and Mild temperatures generate the highest rental activity.
- Moderate and Humid conditions show strong demand.
- Most records fall into the Medium and High demand categories.
- Peak demand reached 249 rentals in a single record.
